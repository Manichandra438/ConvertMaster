# Brain Index — ConvertMaster

## Quick Facts
- **What it is:** Browser-only dev-tool + file-conversion suite (Base64, URL, JSON, JWT, HTML entities, Image converter, PDF tools, Resize/Compress). No server, no uploads — all conversion runs client-side.
- **Stack:** React 19 + Vite 7, react-router-dom 7, framer-motion, Tailwind CSS 3, pdf-lib + pdfjs-dist for PDF, lucide-react icons, react-helmet-async for SEO.
- **Entry points:** `src/main.jsx` → `src/App.jsx` (routes) → `src/components/Layout.jsx` (VSCode-style shell: titlebar/sidebar/statusbar).
